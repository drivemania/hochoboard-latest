# HOCHOBOARD

HOCHOBOARD는 비툴커뮤니티 사용자를 위한 CMS 게시판입니다.  
PHP Slim Framework 4와 Laravel Eloquent ORM을 기반으로 제작되었습니다.  
Blade 템플릿 엔진 및 Alpine.js의 개발 구조를 따릅니다.

## 🖥️ 권장 사양 (Requirements)

HOCHOBOARD를 설치하고 운영하기 위한 서버 요구 사항입니다.

* **PHP:** 8.0 이상
* **Database:** MySQL 5.7+ 또는 MariaDB 10.3+
* **Web Server:** Apache 또는 Nginx
* **PHP Extensions:**
    * `json`
    * `pdo`, `pdo_mysql`
    * `mbstring`
